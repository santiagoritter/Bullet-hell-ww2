Si a la muerte quieres burlar en este infierno de metal,
un código muy antiguo deberás teclear:

Da dos pasos mirando al cielo,
y dos pasos mirando al suelo.
De izquierda a derecha muévete con destreza,
y de izquierda a derecha repite la proeza.

Toca las letras B y A para el gran final,
¡y ninguna bala de este mundo te podrá hacer mal
